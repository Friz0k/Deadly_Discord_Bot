## 🎯 Что делает бот?

**GTA5 RP Assistant** — это мощный Discord‑бот, который **автоматизирует рутинные операции** и **ведёт точный учёт** для вашей ролевой семьи.  
Он управляет семьёй, общим складом, семейным банком, автопарком, контрактами, дисциплинарной системой, модерацией и логирует серверные события в реальном времени.

Все данные хранятся в **SQLite** и могут быть **сохранены/восстановлены** одной командой.

---

## ✨ Ключевые возможности

- 👥 **Управление семьёй**  
  Добавление/удаление участников, привязка Discord к игровому нику, авто‑добавление через роль.  
  *Автосинхронизация ников каждые 10 минут.*

- 💰 **Общий семейный банк**  
  Пополнение/снятие средств, просмотр баланса. Все операции логируются.

- 📦 **Склад с категориями**  
  Предметы разделены на **Оружие**, **Патроны**, **Расходники** и **Прочее**.  
  Визуальные индикаторы запасов (зелёный / жёлтый / красный) и общий счётчик.

- 🚗 **Автопарк**  
  Учёт машин, закрепление за членами семьи, временная выдача с автоматическим возвратом по истечении времени.

- 📝 **Контракты**  
  Создание контрактов с несколькими участниками, дедлайном и наградой.  
  Ежечасные напоминания, автоматическое завершение по истечении срока, уведомления в отдельные каналы.

- ⚠️ **Дисциплинарная система**  
  Выдача предупреждений / выговоров (массовая или индивидуальная), автоматическая конвертация 2 предупреждений → выговор.  
  Роли для каждого уровня наказания, авто‑снятие через 7 дней, защита отпуска.

- 🛡️ **Модерация слов**  
  Автоматическое удаление сообщений с запрещёнными словами и логирование инцидента.

- 📋 **Полное логирование сервера**  
  Смена ника, изменение ролей, перемещения по голосовым каналам, удалённые сообщения, баны/кики — всё в специальном канале.

- 💾 **Бекап и восстановление**  
  Скачайте файл базы данных перед перезапуском и загрузите его обратно, чтобы сохранить данные.

- 🔄 **Текстовые дублёры**  
  Все критически важные команды работают как через слеш (`/`), так и через классический префикс `!`.

## 👑 Роли и доступ

| Роль Discord | Разрешённые действия |
|--------------|----------------------|
| **Тех. Состав** (суперадмин) | Все команды |
| **Recruiter** | Управление семьёй |
| **Assistant** | Автомобили, пополнение склада, баланс банка, бекапы |
| **Deadly** | Финансы, изъятие со склада, выдача/возврат авто, контракты |
| **Discipline** | Дисциплинарные взыскания |
| **Отпуск** | Иммунитет к дисциплинарным взысканиям |

## 📋 Команды

### 👥 Семья
| Команда | Описание |
|---------|----------|
| `/дсемья ID Ник` | Добавить участника |
| `/усемья ID` | Удалить участника |
| `/семья` | Список семьи |
| `/id @user` | Узнать Discord ID |

### 💰 Банк
| Команда | Описание |
|---------|----------|
| `/банк` | Показать баланс |
| `/пополнить Сумма [Причина]` | Пополнить счёт |
| `/снять Сумма [Причина]` | Снять средства |

### 📦 Склад
| Команда | Описание |
|---------|----------|
| `/склад [Категория]` | Просмотр содержимого |
| `/псклад Предмет Категория Кол-во` | Положить предмет |
| `/всклад Предмет Кол-во` | Взять предмет |

### 🚗 Авто
| Команда | Описание |
|---------|----------|
| `/давто Модель Госномер` | Добавить автомобиль |
| `/уавто Госномер` | Удалить автомобиль |
| `/авто` | Список машин |
| `/взавто Номер [Часы]` | Выдать авто (по умолч. 2 ч) |
| `/веавто Номер` | Вернуть авто |

### 📝 Контракты
| Команда | Описание |
|---------|----------|
| `/вк @Участники Название ДД.ММ.ГГГГ ЧЧ:ММ [Векселя]` | Создать контракт |

### ⚠️ Дисциплина
| Команда | Описание |
|---------|----------|
| `/дв @Участники Тип Причина` | Выдать взыскание |
| `/выг [@Участник]` | Просмотр выговоров |
| `/снятьдв @Участник Причина` | Снять последнее взыскание |

### 📋 Логи и бекап
| Команда | Описание |
|---------|----------|
| `/logs [@Участник]` | Посмотреть логи действий |
| `/backup` | Скачать базу данных |
| `/restore` | Загрузить базу данных |
| `/reset_contracts` | Исправить таблицу контрактов |
| `/хелп` | Помощь по боту |

> 💡 **Предпочитаете текстовые команды?**  
> Все команды также работают с префиксом `!`: `!банк`, `!пополнить`, `!дв` и т.д.
