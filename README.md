# 🚗 GTA5 RP Assistant — Discord Bot

Многофункциональный бот для управления фракцией на сервере GTA 5 RolePlay.  
Семья, склад, банк, контракты, дисциплинарная система, модерация и логирование серверных событий.

---

## 📌 Возможности

- 👥 **Управление семьёй** — добавление/удаление участников, привязка Discord к игровому нику, авто-добавление через роль.
- 💰 **Семейный банк** — пополнение и снятие средств, просмотр баланса, логирование операций.
- 📦 **Склад с категориями** — добавление/изъятие предметов, разделение по категориям (Оружие, Патроны, Расходники, Прочее), индикация запасов.
- 🚗 **Автопарк** — учёт автомобилей, выдача и возврат, автоматическое освобождение по истечении времени.
- 📝 **Контракты** — создание контрактов с участниками, дедлайном и векселями, напоминания каждый час, автоматическое завершение, уведомления в отдельный канал.
- ⚠️ **Дисциплинарная система** — выдача предупреждений/выговоров (в т.ч. массово), автоматическая конвертация 2 предупреждений в выговор, роли за наказания, автоматическое снятие через 7 дней, защита от выдачи в отпуске.
- 🛡️ **Модерация слов** — удаление сообщений с запрещёнными словами, логирование инцидентов.
- 📋 **Логирование** — запись действий бота, изменений ролей, смены ника, перемещений по голосовым каналам, удаления сообщений, банов/киков.
- 💾 **Бекап и восстановление базы данных** — команды `!backup` и `!restore` для сохранения данных между деплоями.
- 🔄 **Автообновление ников** — каждые 10 минут синхронизация игровых имён с серверными никнеймами.

## 👑 Роли и доступ

| Роль | Назначение |
|------|------------|
| **Тех. Состав** | Полный доступ ко всем командам |
| **Recruiter** | Управление семьёй (добавление, удаление, просмотр) |
| **Assistant** | Добавление/удаление авто, пополнение склада, просмотр баланса, бекап |
| **Deadly** | Финансы (пополнение/снятие), склад (просмотр/изъятие), авто (просмотр/выдача/возврат), контракты |
| **Discipline** | Дисциплинарные взыскания (выдача, снятие, просмотр) |
| **Отпуск** | Иммунитет к выдаче ДВ |

## 📋 Команды

### 👥 Семья
| Команда | Описание | Доступ |
|---------|----------|--------|
| `/дсемья ID Ник` | Добавить участника в семью | Recruiter |
| `/усемья ID` | Удалить участника из семьи | Assistant |
| `/семья` | Показать список семьи | Recruiter |
| `/id @user` | Узнать Discord ID | Recruiter |

### 💰 Банк
| Команда | Описание | Доступ |
|---------|----------|--------|
| `/банк` | Баланс семьи | Assistant |
| `/пополнить Сумма [Причина]` | Пополнить счёт | Deadly |
| `/снять Сумма [Причина]` | Снять средства | Deadly |

### 📦 Склад
| Команда | Описание | Доступ |
|---------|----------|--------|
| `/склад [Категория]` | Просмотр склада (все/по категориям) | Deadly |
| `/псклад Предмет Категория Кол-во` | Положить предмет | Assistant |
| `/всклад Предмет Кол-во` | Взять предмет | Deadly |

### 🚗 Авто
| Команда | Описание | Доступ |
|---------|----------|--------|
| `/давто Модель Госномер` | Добавить автомобиль | Assistant |
| `/уавто Госномер` | Удалить автомобиль | Assistant |
| `/авто` | Список всех авто | Deadly |
| `/взавто Номер [Часы]` | Выдать авто (по умолчанию 2 ч) | Deadly |
| `/веавто Номер` | Вернуть авто | Deadly |

### 📝 Контракты
| Команда | Описание | Доступ |
|---------|----------|--------|
| `/вк @Участники Название ДД.ММ.ГГГГ ЧЧ:ММ [Векселя]` | Создать контракт | Deadly |

### ⚠️ Дисциплина
| Команда | Описание | Доступ |
|---------|----------|--------|
| `/дв @Участники Тип Причина` | Выдать взыскание | Discipline |
| `/выг [@Участник]` | Показать выговоры | Discipline |
| `/снятьдв @Участник Причина` | Снять последнее взыскание | Discipline |

### 📋 Логи и бекап
| Команда | Описание | Доступ |
|---------|----------|--------|
| `/logs [@Участник]` | Посмотреть логи действий | Assistant |
| `/backup` | Сохранить базу данных в файл | Assistant |
| `/restore` | Восстановить базу из файла | Assistant |
| `/reset_contracts` | Исправить таблицу контрактов | Assistant |
| `/хелп` | Помощь по боту | Все |

> **Текстовые дублёры:** для тех, у кого не работают слеш-команды, доступны такие же команды с префиксом `!` (например, `!банк`, `!пополнить`, `!дв`).
